Capitulo 2: Ambiente de Desarrollo en Windows

2.1 Instalar Ruby, Git y Node

En la terminal de Windows:


	node -v


	npm -v


	gem -v

2.2 Instalando Gems

En la terminal de Windows:


    npm - v


    npm install -g bower grunt-cli


    gem -v


    gem install foundation


    foundation


	gem install sass


	sass -v


	gem install compass


	gem install bundler

2.3 Creando Proyecto en Windows

En la terminal de Windows:


	cd Desktop


	foundation new hotel 


	bundle


	bundle exec compass watch

2.4 Instalando RubyDevKit

En la terminal de Windows:


	cd RubyDevKit


	ruby dk.rb init


	ruby dk.rb install


	gem update

Capitulo 3: Ambiente de Desarrollo en Windows

3.1 Instalar Git y Node

En la terminal de OSX:


    node -v


    npm -v


    git

3.2 Instalando Gems

En la terminal de OSX:


	gem -v


	sudo gem update --system


	sudo npm install -g bower grunt-cli


	sudo gem install foundation


	foundation 


	sudo gem install sass


	sass -v


	sudo gem install compass


	compass -v


	sudo gem install bundler


	bundler -v

3.3 Creando Proyecto

En la terminal de OSX:


	foundation


	pwd


	ls


	cd Desktop


	foundation new hotelWebSite


	cd hotelWebSite


	bundle


	bundle exec compass watch

3.4 Como actualizar gems desde OSX

En la terminal de OSX:



	sudo gem update


	foundation
